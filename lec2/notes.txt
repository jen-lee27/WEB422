1. Use Bootstrap to create responsive, dynamic web pages. Use CDN to include bootsrap into page.

Using Emmet, please "!" in vscode to generate basic structure of the web page.

2. Paste the following two lines in the head of the web page which was generated by emmet

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.1/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-4bw+/aepP/YC94hEpVNVgiZdgIC5+VKNBQNGCHeKRQN+PtmoHDEXuppvnDJzQIu9" crossorigin="anonymous">
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.1/dist/js/bootstrap.bundle.min.js" integrity="sha384-HwwvtgBNo3bZJJLYd8oVXjrBZt8cqVSpeBNS5n7C8IVInixGAoxmnlMuBnhbgrkm" crossorigin="anonymous"></script>

Run program in the browser to see everything is working.

3. Containers help in creating Grids in Bootstrap. Use the following to create a simple container with border.

In vscode, use .container to create a div class

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.1/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-4bw+/aepP/YC94hEpVNVgiZdgIC5+VKNBQNGCHeKRQN+PtmoHDEXuppvnDJzQIu9" crossorigin="anonymous">
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.1/dist/js/bootstrap.bundle.min.js" integrity="sha384-HwwvtgBNo3bZJJLYd8oVXjrBZt8cqVSpeBNS5n7C8IVInixGAoxmnlMuBnhbgrkm" crossorigin="anonymous"></script>
    <title>Document</title>
</head>
<body>
    <div class="container border">Seneca College</div>
</body>
</html>

4. Update class with container-fluid to make it take the full width

<div class="container-fluid border">Seneca College</div>

5. In order to add rows and columns, we can use row and col inside the container along with a style provided

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.1/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-4bw+/aepP/YC94hEpVNVgiZdgIC5+VKNBQNGCHeKRQN+PtmoHDEXuppvnDJzQIu9" crossorigin="anonymous">
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.1/dist/js/bootstrap.bundle.min.js" integrity="sha384-HwwvtgBNo3bZJJLYd8oVXjrBZt8cqVSpeBNS5n7C8IVInixGAoxmnlMuBnhbgrkm" crossorigin="anonymous"></script>
    <title>Document</title>
    <style>
        [class *= "col"]{
            border: 1px solid #000;
        }
    </style>
</head>
<body>
    <div class="container border">
        <div class="row">
            <div class="col">Column 1</div>
            <div class="col">Column 2</div>
        </div>
    </div>
</body>
</html>

6. Specific spaces can be given to the columns as well

<div class="container border">
        <div class="row">
            <div class="col-4">Column 1</div>
            <div class="col-4">Column 2</div>
            <div class="col">Column 3</div>
        </div>
    </div>

7. Breakpoints can be provided as well to ensure that columns move to new row if screen size is too small

    <div class="container border">
        <div class="row">
            <div class="col-md-4">Column 1</div>
            <div class="col-md-4">Column 2</div>    
        </div>
    </div>

8. Row alignments are also possible

    <div class="container border">
        <div class="row align-items-center" style="height: 300px;">
            <div class="col-md-4">Column 1</div>
            <div class="col-md-4">Column 2</div>    
            <div class="col-md-4">Column 3</div>    
        </div>
    </div>

9. Rows can be horizontally aligned as well

    <div class="container border">
        <div class="row justify-content-center">
            <div class="col-md-4">Column 1</div>
            <div class="col-md-4">Column 2</div>       
        </div>
    </div>

10. Buttons are important and can be created as below:
    <div class="container">
        <button class="btn btn-primary">Login</button>
    </div>

Buttons can be styled for changing size and disabling them

<button class="btn btn-primary btn-lg" disabled>Login</button>

                <a class="nav-link" href="#">Link</a>
              </li>

                  Dropdown
